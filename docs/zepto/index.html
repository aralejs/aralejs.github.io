Zepto

Zepto 是一个非常小巧的适用于现代浏览器的 JavaScript 类库。Zepto 的 API 绝大部分与 jQuery 兼容。如果你会使用 jQuery，你也就会用 Zepto.

模块依赖

  • seajs

使用说明

define(function(require, exports) {

    var $ = require('zepto');

    // 快乐使用 $ 吧

});

详细 API, 请阅读精心编写的 Zepto 官方文档：http://zeptojs.com/

更新

当 Zepto 发布新版本，需要更新时，只要运行：

$ cd arale/dist
$ spm install zepto
