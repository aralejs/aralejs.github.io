Handlebars

Handlebars 是一个轻量级的模板引擎。它是 Mustache 的超集，支持更多特性，但保持了 logicless 特质。

模块依赖

  • seajs

使用说明

精心书写的官方文档：handlebarsjs.com/

很棒的在线试用页面：tryhandlebarsjs.com/

更新

当有新版本发布，需要更新时，请运行：

$ cd arale/dist
$ spm install handlebarsjs
