
# Popup

Popup 是可触发 Overlay 型 UI 组件。

---


## 依赖

 - [seajs](seajs/README.md)
 - [position](position/README.md)
 - [overlay](overlay/README.md)


## 使用说明


### Dropdown

```
seajs.use(['jquery','dropdown'], function($, Dropdown){
    var example = new Dropdown({
        trigger: '#triggerId',
        element: '#targetId'
    });
});
```
